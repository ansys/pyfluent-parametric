Introduction
------------
PyFluent-Parametric provides scripting of a parametric study using Ansys Fluent.
It is part of the `PyAnsys <https://docs.pyansys.com>`_ ecosystem that
lets you use Fluent within or alongside any other Python environment,
whether it is in conjunction with other Ansys Python libraries and packages or
with other external Python products.

Documentation and issues
------------------------
In addition to installation and usage information, the PyFluent-Visualization
documentation provides :ref:`API reference`, :ref:`Examples`, and :ref:`Contributing`
sections.

On the `PyFluent-Parametric Issues <https://github.com/pyansys/pyfluent-parametric/issues>`_,
you can create issues to submit questions, report bugs, and request new features. To reach
the PyAnsys support team, email `pyansys.support@ansys.com <pyansys.support@ansys.com>`_.
